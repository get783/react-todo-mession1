# 2주차 팀 활동: 리액트 투두 앱 만들기

## 전체 기능

| 기능 | 설명 |
| --- | --- |
| ✅ 할 일 추가  | 입력 후 버튼 클릭으로 추가    |
| ❌ 할 일 삭제  | 삭제 버튼 클릭으로 삭제      |
| ✔ 완료 체크   | 체크박스 클릭으로 완료 표시    |
| 💾 데이터 저장 | Local Storage 자동 저장 |
| 🔄 초기 로드  | 새로고침 시 데이터 유지      |
